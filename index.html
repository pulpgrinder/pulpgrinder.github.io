Fronkensteen

30.12.2020 08:44
Fronkensteen is a self-contained, single-file, fully-programmable wiki-like environment that works in any full-featured modern browser (i.e., just about everything other than Internet Explorer or pre-Chrome Edge). You can write and execute apps without any external resources at all. No toolchain. No server. No external editor. The entire system is contained in a single HTML file which can be opened in a browser from the local file system, put on a standard web server, embedded in a native wrapper (such as Electron or Cordova).
